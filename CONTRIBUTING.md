# Contributing to Frosten IDE

Thank you for your interest in contributing! ❄️

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/frosten-ide.git`
3. Install dependencies: `npm install` 
4. Start dev mode: `npm run dev`

## Branch Naming

| Type | Format |
|---|---|
| Feature | `feat/feature-name` |
| Bug fix | `fix/bug-description` |
| Docs | `docs/what-you-changed` |
| Refactor | `refactor/what-changed` |

## Commit Style (Conventional Commits)
```
feat: add tab auto-close on file delete
fix: resolve Monaco editor blank on reload
docs: update provider setup table
```

## What to Contribute

- New AI provider presets (Settings panel)
- File tree performance improvements
- New editor themes
- Agent step types (browser automation, API calls, etc.)
- Bug fixes listed in [Issues](https://github.com/indianicsoft/frosten-ide/issues)

## Pull Request Checklist

- [ ] `npm run dev` runs without errors
- [ ] No hardcoded API keys or credentials
- [ ] Code follows existing structure
- [ ] PR description explains what and why
